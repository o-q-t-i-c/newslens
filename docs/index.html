・色付きの言葉は、その媒体グループ(新聞・通信・放送/ウェブメディア)だけが見出しに使った語です。機械抽出のため、単なる言い回しの違いが含まれることもあります。
・グレーの媒体は「このフィードでは未確認」の意味で、報道していないと断定するものではありません(RSSは各社の記事の一部のみを含みます)。
・スポーツ・芸能・天気予報は扱いません。試合結果や気温のような事実報道には、報じ方の差が構造的に生まれないためです。
・見出しと画像は各媒体の公開情報(RSS/OGP)のみを使用し、本文は転載しません。クリックすると各媒体の記事ページが開きます。
